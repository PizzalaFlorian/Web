Question 1

Le code JavaScript s'exécute :

Dans le navigateur
Sur le serveur où est stocké le code
Tantôt dans le navigateur, tantôt sur le serveur

Le code JavaScript s'exécute toujours sur les clients (ordinateurs, tablettes et téléphones) qui font référence à ce code via une page HTML.

Question 2

Lorsque l'on veut exécuter du code JavaScript, attendre la disponibilité du DOM est :

Vital
Inutile
Parfois important, parfois sans importance

Il est impératif d'attendre la disponibilité du DOM avant d'exécuter du code JavaScript. Sans quoi, ce code pourrait s'appliquer à un élément indisponible et provoquer un comportement inattendu, voire même un plantage du navigateur.

Question 3

Quel énoncé décrit correctement la relation entre JavaScript et Java ?

JavaScript et Java ne sont que des noms différents pour la même chose
Les deux langages apportent de l'interactivité aux pages Web
Les codes JavaScript et Java sont compilés vers du ByteCode, qui est interprété par le navigateur

JavaScript et Java sont des langages différents. Le code source JavaScript est interprété, alors que le code Java est compilé. Java peut être intégré sous forme d'applet à des pages web. La bonne réponse est donc la deuxième.
